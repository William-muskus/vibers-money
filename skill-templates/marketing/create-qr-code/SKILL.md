# Create QR Code (Marketing / CMO)

## When to use
- You need a **QR code** that links to a website or page (e.g. for events, flyers, business cards). Use the URL provided by your mission or CEO.

## How to use

1. **Domain access**  
   Ensure **`qrcodecreator.com`** is in your approved domains for Computer Use. If not, request access from the Security Director via Swarm Bus.

2. **Open the generator**  
   Using the Computer Use MCP:
   - **Navigate** to: **https://qrcodecreator.com/**
   - **Wait** for the page to load, then take a **screenshot** to see the interface.

3. **Choose type and enter URL**  
   - The site offers types (Website, PDF, Email, Social media, etc.). For a **website link**, choose **Website** (or "Add Website Link").
   - **Click** the URL/website input field and **type** the target URL from your brief or mission.
   - **Screenshot** to confirm the preview of the QR code updates.

4. **Design (optional)**  
   - You can customize with colors, logos, or shapes if needed. For a quick demo, the default is fine.
   - If you need to download, you may need to sign in (the site may offer a free trial or sign-up). Follow the on-screen prompts with **click** and **type** as needed.

5. **Download**  
   - **Click** the **Download** or **Generate** button (e.g. "Download QR", "Generate svg", or similar — use the element list from your **screenshot** to find the correct button).
   - If a sign-up or login is required to download, use the **solve-captcha** skill if a captcha appears; otherwise complete the form with Computer Use.
   - Save the file to your workspace (e.g. `drafts/qr-vibers.png` or similar) if your environment supports it, or note where it was downloaded.

6. **Report**  
   Inform the CEO or requester via Swarm Bus that the QR code was created and where it points.

## Convention
- Use the URL from your mission or CEO. One QR per purpose is enough unless asked for multiple.
