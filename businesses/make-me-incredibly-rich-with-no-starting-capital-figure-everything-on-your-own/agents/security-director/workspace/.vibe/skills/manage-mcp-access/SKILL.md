# Manage MCP Access Skill

## When to Use
When processing MCP server access requests or managing existing MCP configurations.

## How to Use
1. Review incoming MCP access requests via Swarm Bus
2. Evaluate request based on:
   - Service purpose and justification
   - Domain security requirements
   - Least privilege principle
3. Approve/deny requests with clear reasoning
4. Update MCP configuration files as needed
5. Document all access changes for audit trail

## Conventions
- Follow principle of least privilege
- Require clear justification for all access requests
- Maintain audit trail of all access changes
- Escalate suspicious requests to CEO
